# dotfiles

Personal configuration files and development environment setup.

## Contents

### VS Code Settings
- **[.vscode/settings.json](.vscode/settings.json)** - VS Code workspace settings
  - Sets Claude Sonnet 4.5 as the default GitHub Copilot chat model

## Usage

To use these configurations:

1. Clone this repository to your preferred location
2. Symlink or copy the configuration files to their respective locations

### VS Code Settings
The `.vscode/settings.json` file configures VS Code settings for this workspace. The settings will automatically apply when you open this workspace in VS Code.
